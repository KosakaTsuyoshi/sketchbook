Classpublic class URLLoaderHelper

URLLoaderを簡略化したクラスです。

URLLoaderのイベントに対するコールバックハンドラや、URLRequestを介さず直接URLをコールするメソッドを提供します。

TODO: Flashのundocumentな使用か、onCompleteというハンドラがうまく動かない


Example 
  var loader:URLLoaderSB = new URLLoader()
  loader.onComplete=function(data:Object):void
  {
    trace("loadCompleted")  
  }
  loader.loadURL("http://www.fladdict.net")

loadURL()method 
public function loadURL(url:String, params:Object = null, method:String = "GET", headerParams:Object = null):void

URLRequestを介さずに、URLをダイレクトに指定するロード方法です。

Parameters
url:String — ロードするURL
 
params:Object (default = null) — サーバーに渡す引数を格納した無名オブジェクト
 
method:String (default = "GET") — POST or GET
 
headerParams:Object (default = null) — ヘッダーとして送信する値を格納した無名オブジェクト

Example 
   var params:Object = {keyword:"test", sort:"asc"}
      var loader:URLLoaderSB = new URLLoader()
      loader.loadURL("http://www.fladdict.net",params,"POST")
