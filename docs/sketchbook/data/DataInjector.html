Classpublic class DataInjector

DataInjectorはオブジェクトに、複数のプロパティを注入するクラスです。

XMLによって注入するプロパティのグループを定義することで、 コード内の全てのパブリックな、String, Number, Boolean肩の変数を外出しすることができます。

将来的にはObjectやArray、Date型、クラス参照等も注入可能にする予定
